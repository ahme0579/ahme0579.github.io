HTML Tags

Emphasis tag
Defines text with slightly more significant content. For this content, screen readers will switch to a different voice.
Strong tag
Defines text with most importance. Screen readers will change their voice for this
Abbreviation tag (Abbr)
Defines an abbreviation or an acronym
Citation tag
Defines the title of a creative work
Quotation tag
Defines a short quotation
